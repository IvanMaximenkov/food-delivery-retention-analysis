## Текущий прогресс

По состоянию на 23 апреля 2026 года в проекте завершен этап первичного исследовательского анализа данных

### Первые выводы
- Активность пользователей распределена неравномерно: относительно небольшое ядро более активных клиентов формирует заметную долю заказов.
- Ранние поведенческие сигналы показывают, что паттерны повторных заказов, вероятно, станут основной retention-метрикой на следующем этапе анализа.
- Транзакционные и сессионные признаки уже на этом этапе выглядят полезными для будущей модели оттока.
- Этап exploratory analysis помогает зафиксировать baseline-метрики, которые далее будут использоваться в retention, сегментации и A/B-анализе.
