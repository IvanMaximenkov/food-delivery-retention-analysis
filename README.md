# Продуктовый анализ retention и предсказание оттока в приложении доставки еды

## Описание проекта
Проект представляет собой полноценный аналитический кейс для приложения доставки еды

Я провёл глубокий продуктовый анализ **retention** пользователей, выявил ключевые драйверы оттока (churn) и построил ML-модель для предсказания вероятности оттока. Проект включает работу с синтетическими данными, SQL, когортный анализ, RFM-сегментацию, полный цикл A/B-тестирования и интерпретируемую модель машинного обучения.

## Бизнес-проблема
В foodtech одной из главных проблем является **низкий retention** пользователей: многие клиенты делают 1–2 заказа и перестают пользоваться приложением. Это приводит к высоким затратам на привлечение новых пользователей (CAC) и упущенной выручке.

## Цель проекта
- Глубокий анализ продуктовых метрик (Retention, Churn, ARPU, AOV, LTV)
- Выявление инсайтов и формулирование actionable бизнес-рекомендаций с расчётом эффекта в рублях
- Построение предсказательной ML-модели churn с интерпретируемостью (SHAP)
- Полный цикл работы от генерации данных до презентации результатов
