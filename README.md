## Текущий прогресс

По состоянию на 26 апреля 2026 года в проекте завершен этап первичного исследовательского анализа данных и добавлен когортный retention-анализ, сфокусированный на повторных заказах после первой покупки

### Первые выводы
- Активность пользователей распределена неравномерно: относительно небольшое ядро более активных клиентов формирует заметную долю заказов.
- Ранние поведенческие сигналы показывают, что паттерны повторных заказов, вероятно, станут основной retention-метрикой на следующем этапе анализа.
- Транзакционные и сессионные признаки уже на этом этапе выглядят полезными для будущей модели оттока.
- Этап exploratory analysis помогает зафиксировать baseline-метрики, которые далее будут использоваться в retention, сегментации и A/B-анализе.

### Что показывает когортный retention-анализ
- retention логично оценивать через повторный заказ в фиксированное окно после первого заказа;
- когортный анализ позволяет увидеть различия в удержании между группами пользователей, пришедшими в разные периоды;
- результаты этого этапа будут использоваться дальше в сегментации пользователей и churn modeling.

## Следующий этап: RFM-сегментация

Следующий шаг в проекте — сегментация пользователей по модели RFM (`Recency`, `Frequency`, `Monetary`). Этот этап нужен, чтобы перейти от общего retention-анализа к более прикладному пониманию клиентской базы и выделить группы пользователей с разной бизнес-ценностью.

### Зачем нужен этот этап
- определить наиболее ценные сегменты пользователей и понять, за счет кого формируется основная выручка;
- выделить клиентов с высоким риском оттока, но с потенциалом для возврата;
- отделить активных, спящих и low-value пользователей для дальнейших retention-механик;
- использовать сегменты как основу для продуктовых гипотез, таргетированных коммуникаций и будущей churn-модели.
